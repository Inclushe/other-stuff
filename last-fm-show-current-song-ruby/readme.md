# Last.fm Show Current Song

Simple program that shows a [last.fm](http://last.fm/) user's currently playing / last played song.

![alt text](http://i.imgur.com/H3TrLH8.png)

## Prerequisites

Ruby (tested on 2.2.1 and 2.2.3)

## Usage
    ruby lastfm

The program automatically saves the last user you've entered in.

You can change the username entered by typing:

    ruby lastfm -u *ENTER USERNAME HERE*
    ruby lastfm -username *ENTER USERNAME HERE*
    ruby lastfm --username *ENTER USERNAME HERE*
    ruby lastfm whatever else
